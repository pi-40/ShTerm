# ShTerm
You can now use Bash on Windows. Introducing Shterm: you can now use Bash commands, and it even has its own command, CTH. GitHub link lets you download GitHub packages/releases inside the terminal.

> [!WARNING]
> Executing random Bash files could leave your computer at risk. It is important that you avoid unknown sources and unverified Bash files. Do not use the `cth` command on repositories you don't know; doing so will leave your computer at risk.
